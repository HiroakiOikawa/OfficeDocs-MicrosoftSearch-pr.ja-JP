### <a name="step-1-create-the-default-associations-file"></a><span data-ttu-id="966f4-109">手順 1: 既定の関連付けファイルの作成</span><span class="sxs-lookup"><span data-stu-id="966f4-109">STEP 1: Create the default associations file</span></span>
<span data-ttu-id="966f4-110">ドメイン コントローラーの SYSVOL フォルダーに既定の関連付けファイルを作成します。</span><span class="sxs-lookup"><span data-stu-id="966f4-110">These steps try and create the default associations file in the SYSVOL folder of the domain controller.</span></span>

1. <span data-ttu-id="966f4-111">管理者 PowerShell コンソールを開きます。</span><span class="sxs-lookup"><span data-stu-id="966f4-111">Open an administrative PowerShell console.</span></span>
1. `New-Item -Path "\\$env:USERDOMAIN\SYSVOL\$env:USERDNSDOMAIN" -Type Directory -Name "Settings"`
1. `$SettingsPath="\\$env:USERDOMAIN\SYSVOL\$env:USERDNSDOMAIN\Settings"`
1. `Start-Process Dism.exe -PassThru "/Online /Export-DefaultAppAssociations:$SettingsPath\AppAssoc.xml"`
    
  
### <a name="step-2-add-or-edit-the-default-associations-file"></a><span data-ttu-id="966f4-112">手順 2.</span><span class="sxs-lookup"><span data-stu-id="966f4-112">Step 2</span></span> <span data-ttu-id="966f4-113">既定の関連付けファイルの追加または編集</span><span class="sxs-lookup"><span data-stu-id="966f4-113">Add or edit the default associations file</span></span>

1. `Notepad "$SettingsPath\AppAssoc.xml"`

    - `<Association Identifier=".htm" ProgId="AppX4hxtad77fbk3jkkeerkrm0ze94wjf3s9" ApplicationName="Microsoft Edge" />`
              
    - `<Association Identifier=".html" ProgId="AppX4hxtad77fbk3jkkeerkrm0ze94wjf3s9" ApplicationName="Microsoft Edge" />`
    - `<Association Identifier="http" ProgId="AppXq0fevzme2pys62n3e0fbqa7peapykr8v" ApplicationName="Microsoft Edge" />`
    
  - <span data-ttu-id="966f4-116">**Internet Explorer**</span><span class="sxs-lookup"><span data-stu-id="966f4-116">**Internet Explorer**</span></span>

    - `<Association Identifier="http" ProgId="IE.HTTP" ApplicationName="Internet Explorer" />`
