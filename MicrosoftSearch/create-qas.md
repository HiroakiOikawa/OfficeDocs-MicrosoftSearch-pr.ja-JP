# <a name="create-qas"></a><span data-ttu-id="b71cc-103">Q&A の作成</span><span class="sxs-lookup"><span data-stu-id="b71cc-103">Create Q&As</span></span>

<span data-ttu-id="b71cc-p101">Q&A では、ユーザーに対する回答や情報を、必要に応じてリンクを含めて提供します。理想的には、Q&A には、ユーザーが情報源に移動しなくても済むよう探している詳細をすべて含めます。Q&A のコンテンツの書式を設定し、画像、リスト、表などを含めることができます。</span><span class="sxs-lookup"><span data-stu-id="b71cc-p101">Q&As provide an answer or info to users including an optional link. ## <a name="create-a-qa"></a><span data-ttu-id="b71cc-107">Q&A を作成する</span><span class="sxs-lookup"><span data-stu-id="b71cc-107">Create a Q&A</span></span>

<span data-ttu-id="b71cc-108">効果的なタイトル、説明、キーワードなどの作成方法については、「[コンテンツの計画](plan-your-content.md)」をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="b71cc-108">For information about creating effective titles, descriptions, keywords, and more, see [Plan your content.](plan-your-content.md)</span></span>
  
1. <span data-ttu-id="b71cc-109">Microsoft Search 管理ポータルに移動します</span><span class="sxs-lookup"><span data-stu-id="b71cc-109">Go to the Microsoft Search Admin portal</span></span>
    
2. <span data-ttu-id="b71cc-110">ナビゲーション ウィンドウで、**[Q&A]** をクリックします</span><span class="sxs-lookup"><span data-stu-id="b71cc-110">In the left navigation pane, click **Settings**.</span></span>

    <span data-ttu-id="b71cc-p102">[Q&A の編集] ページに、Bing で Q&A がどのように表示されるかがプレビューされます。必要な情報を追加すると、自動的にプレビューが更新されます。</span><span class="sxs-lookup"><span data-stu-id="b71cc-p102">The Edit Q&A page appears, with a preview of how the Q&A will look on Bing. 4. <span data-ttu-id="b71cc-114">**タイトル**を入力します</span><span class="sxs-lookup"><span data-stu-id="b71cc-114">Enter a **Title** for the new tile.</span></span>
    
    <span data-ttu-id="b71cc-p103">タイトルは、結果に表示される見出しです。最大 120 文字まで可能で、質問形式にすることをお勧めします。</span><span class="sxs-lookup"><span data-stu-id="b71cc-p103">The title is the heading that appears in the result. It can be up to 120 characters long and we recommend using a question format.</span></span>
    
5. <span data-ttu-id="b71cc-117">必要に応じて、ブックマークのリンク先となるページ、サイト、場所の **URL** を入力します</span><span class="sxs-lookup"><span data-stu-id="b71cc-117">If needed, enter the **URL** of the page, site, or location the bookmark will link to</span></span> 
    
    <span data-ttu-id="b71cc-118">この情報によって、ユーザーは詳細を参照する追加の情報源に簡単にアクセスできます。</span><span class="sxs-lookup"><span data-stu-id="b71cc-118">This lets users that need additional information easily get to the source to read more.</span></span>
    
6. <span data-ttu-id="b71cc-119">**回答の説明**を入力します</span><span class="sxs-lookup"><span data-stu-id="b71cc-119">Enter an **Answer description**</span></span>
    
    <span data-ttu-id="b71cc-p104">これは、タイトルで尋ねた質問に対する回答でなければなりません。既存の HTML コンテンツをコピーして貼り付けることができます。サポート対象外のタグはすべて無視されます。</span><span class="sxs-lookup"><span data-stu-id="b71cc-p104">This should answer the question that's asked in the title. 7. <span data-ttu-id="b71cc-123">HTML タグと組み込みオプションを使用して、テキスト、画像、リスト、表などの書式を設定します</span><span class="sxs-lookup"><span data-stu-id="b71cc-123">Use HTML tags and the built-in options to format text, add images, lists, tables, and more</span></span>
    
    <span data-ttu-id="b71cc-p105">ページ上部にあるプレビューを使用して、タグと書籍設定によって Bing でどのように表示されるかを確認します。詳細については以下の資料をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="b71cc-p105">Use the preview at the top of the page to see how your tags and formatting will appear on Bing. For information about:</span></span>

  - <span data-ttu-id="b71cc-127">組み込みオプションについては、**マークダウン ガイド** (疑問符アイコン) をクリックします</span><span class="sxs-lookup"><span data-stu-id="b71cc-127">Built-in options, click **Markdown guide** (question mark icon)</span></span> 
    
8. <span data-ttu-id="b71cc-128">この Q&A をトリガーするための**キーワード**を入力します。</span><span class="sxs-lookup"><span data-stu-id="b71cc-128">Enter **Keywords** you want to trigger this Q&A</span></span> 
    
    <span data-ttu-id="b71cc-p106">ブックマークのように、含めたキーワードをユーザーが検索すると、この Q&A が作業結果に含まれます。Q&A のタイトルに含まれている語句をできるだけ多くのバリエーションで追加してください。</span><span class="sxs-lookup"><span data-stu-id="b71cc-p106">Like a bookmark, when a user searches for any of the keywords you've included, this Q&A will be included in their work results. 9. <span data-ttu-id="b71cc-131">**[類似するキーワードを自動的にマッチさせる]** を選択すると、キーワードの検索範囲を広げられます。</span><span class="sxs-lookup"><span data-stu-id="b71cc-131">Select **Automatically match similar keywords** to expand your set of keywords</span></span> 
    
    <span data-ttu-id="b71cc-132">こうすることで、検索語句が広い範囲でマッチされるようになり、該当する作業結果にこの Q&A が含まれるようになります。</span><span class="sxs-lookup"><span data-stu-id="b71cc-132">This enables a broader matching of search terms and helps ensure this Q&A is included in relevant work results.</span></span>
    
10. <span data-ttu-id="b71cc-133">**予約済みキーワード**を入力します</span><span class="sxs-lookup"><span data-stu-id="b71cc-133">Enter **Reserved keywords**</span></span>
    
    <span data-ttu-id="b71cc-p107">キーワードによって複数の Q&A がトリガーされる場合、Microsoft Search は最もよく参照されるものを最上部に配置し、他のものは関連リンクとして表示します。1 つ以上の予約済みキーワードを使用して、最上位の結果として Q&A が必ず表示されるようにします。予約済みキーワードを複数の Q&A で共有することはできません。また、Q&A とブックマークで予約済みキーワードを共有することは推奨されていません。ブックマークと Q&A でキーワードまたは予約済みキーワードを共有すると、ブックマークが常時優先され、Q&A は表示されません。</span><span class="sxs-lookup"><span data-stu-id="b71cc-p107">If a keyword triggers multiple Q&As, Microsoft Search will place the most popular one at the top and show the others as related links. Also, sharing reserved keywords across Q&As and bookmarks isn't recommended. If a bookmark and a Q&A share a keyword or a reserved keyword, the bookmark will always take precedence and the Q&A will not appear.</span></span>
